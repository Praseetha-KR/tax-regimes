# Tax Regimes Comparison

Compares Indian Tax regimes and suggest which one has the lowest tax.

Supported years are


### Sample Usage

```python
In [1]: from tax_regimes import lowest

In [2]: tax = lowest(1234567, 1500000, 35000, 74000)

In [3]: tax.total_tax
Out[3]: 113997

In [4]: tax.YEAR
Out[4]: 2019

In [5]: print(tax.tabular())
Out[5]:
```
